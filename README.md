# 📖 Math Magicians <a name="about-project"></a>

- Welcome to Math Magicians, a react application for all fans of mathematics. It is a Single Page App (SPA) that allows users to:

    - Make simple calculations.
    - Read a random math-related quote.

## 💻 Getting Started <a name="getting-started"></a>

> Steps to run: 

To get a local copy up and running, follow these steps.

### Prerequisites

In order to clone and run this project you need:

- A working browser
- Installation of Git
- Internet browser
- Code editor like VS code
- Node JS

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/abel-tefera/math-magicians.git
```

### Install

Install this project with:

```sh
  cd math-magicians
  npm install
  npm start
```

### Usage

To run the project, execute the following commands:

- Run `npm start` after installation
- A new browser window should open with the website
